Descarga del proyecto base

Para desarrollar el proyecto final del curso, descargue o clone el proyecto base disponible en el repositorio institucional. Este repositorio servirá como punto de partida para la implementación de los módulos, la configuración de seguridad, la documentación en Swagger y el despliegue con Docker.

Repositorio del proyecto

El proyecto base se encuentra disponible en la siguiente ruta de GitHub:

https://github.com/alejandroria94/proyecto-s3.git

Desde este repositorio podrá obtener la estructura inicial del proyecto, revisar la organización del código y comenzar la construcción del proyecto final a partir de una base común para todos los estudiantes.

¿Qué debe hacer el estudiante?

  1. Ingresar al repositorio del proyecto base en GitHub.
  2. Descargar el proyecto como archivo ZIP o clonarlo con Git.
  3. Abrir la solución en su entorno de desarrollo.
  4. Verificar que el proyecto compile correctamente antes de iniciar modificaciones.
  5. Usar esta base como punto de partida para implementar el proyecto final guiado del curso.
Recomendación: si va a trabajar de forma continua sobre el proyecto, se sugiere clonar el repositorio con Git para facilitar el control de versiones y el seguimiento de los cambios realizados durante el desarrollo.

Comando sugerido para clonar el repositorio

git clone https://github.com/alejandroria94/proyecto-s3.git

Después de clonar el repositorio, ubíquese en la carpeta del proyecto y continúe con la configuración y desarrollo solicitados en la guía del curso.
